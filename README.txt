This code creates an applicant filter program that sets criteria for acceptance. If applicant has a gpa of or over 3.0, a ppersonal statment (graded 1-100) scored 90 or above,and at least 3 extra curricular activities, they will be accepted. If the other criteria are met but there are not enough E.C. activities, they should be brought in for an interview. Otherwise, they should be rejected. After this, there are 4 hypothetical students that are generated using this function.
